$.getJSON(url, data, success);

Les trois paramètres correspondent donc à :

  • le paramètre url, qui est une chaîne contenant l’URL à laquelle la demande est envoyée
  • le paramètre de données facultatif, qui est un objet ou une chaîne envoyée au serveur avec la demande
  • le paramètre facultatif success(data, textStatus, jqXHR), qui est une fonction de rappel exécutée uniquement si la demande réussit

Dans le scénario le plus simple, nous ne nous soucions que de l’objet retourné. Dans ce cas, un rappel de réussite potentiel ressemblerait à ceci :

function success(data) {
// do something with data, which is an object
}

Comme mentionné, la même demande peut être déclenchée avec l’appel plus verbeux $.ajax(). Ici, nous utiliserions:

$.ajax({
dataType: 'json',
url: url,
data: data,
success: success
});

Voyons cela en pratique en faisant une petite démo.
